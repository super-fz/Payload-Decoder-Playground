# Payload-Decoder-Playground
A client-side tool for decoding and analyzing obfuscated payloads commonly found in phishing emails, malware samples, and penetration tests. 

<LI>
Supports multi-layer decoding (Base64, hex, URL, Unicode, etc.)
</LI>

<LI>
Flags suspicious patterns, shell commands
</LI>

<LI>
Indicators of compromise
</LI>


All in your browser with zero data sent to servers.

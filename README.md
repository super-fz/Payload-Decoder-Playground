# Payload-Decoder-Playground
A client-side tool for decoding and analyzing obfuscated payloads commonly found in phishing emails, malware samples, and penetration tests. Whether working on phishing investigations, red teaming, or malware reverse engineering, this tool provides a fast, offline environment to break down and analyze suspicious strings.
